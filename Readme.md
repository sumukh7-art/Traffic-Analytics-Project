# Smart Traffic Analytics and Traffic Density Prediction System

## Project Overview

This project analyzes Delhi traffic data using Data Analytics and Machine Learning techniques. The objective is to understand traffic congestion patterns and predict traffic density levels based on factors such as weather, road type, travel distance, time of day, and average vehicle speed.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook
* Power BI
* Git & GitHub

## Dataset Information

The dataset contains 4,000 traffic records with the following features:

* Trip ID
* Start Area
* End Area
* Distance (km)
* Time of Day
* Day of Week
* Weather Condition
* Road Type
* Average Speed (km/h)
* Traffic Density Level

## Exploratory Data Analysis

Key findings:

* High traffic density was the most common traffic condition.
* Evening Peak experienced the highest congestion levels.
* Main Roads recorded the highest traffic volume.
* Traffic congestion increased as average speed decreased.
* Highways provided the highest average travel speed.

## Machine Learning Model

Algorithm Used:

* Random Forest Classifier

Target Variable:

* Traffic Density Level

Model Accuracy:

* 83.13%

## Feature Importance

1. Average Speed (52.65%)
2. Distance (16.86%)
3. Road Type (10.89%)
4. Time of Day (9.02%)
5. Weather Condition (8.31%)
6. Day of Week (2.26%)

## Conclusion

The project successfully identified important traffic patterns and developed a machine learning model capable of predicting traffic density with 83.13% accuracy.
